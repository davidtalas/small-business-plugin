# Small Business Plugin for Claude Code

111 skills for solopreneurs and small business owners — content creation, email marketing, client management, sales, operations, AI capability guides, and more.

Built by [Dave Talas](https://paladin.cc) for The Paladin Guild community.

## Install

```bash
claude plugin add davidtalas/small-business-plugin
```

That's it. Skills activate automatically when Claude detects a matching task.

## What's included

### Content & Marketing
Email sequences, newsletters, blog posts, social media captions, content calendars, launch announcements, ad copy, landing pages, lead magnets, and content repurposing.

### Sales & Pipeline
Proposals, pricing analysis, sales scripts, pipeline reviews, discovery call prep, lead follow-ups, offer packaging, and win/loss analysis.

### Client Management
Onboarding flows, offboarding checklists, feedback incorporation, results presentations, scope management, and testimonial collection.

### Operations
SOPs, time blocking, weekly reviews, invoicing, project timelines, task breakdowns, and prioritization.

### Contracts & Legal
Contract templates, scope definitions, negotiation prep, terms updates, and handoff plans.

### AI Capability Guides
What ChatGPT, Claude, and Gemini can do. How to use Claude Code, Claude Projects, Custom GPTs, and Gemini Gems. Dave's AI integration framework.

### Make.com Guides
Getting started with Make.com, module capabilities, and automation readiness audits.

## How it works

Skills auto-activate based on what you're doing. Ask Claude to "write my newsletter" and the newsletter skill kicks in. Ask to "create a welcome email sequence" and the email sequence architect takes over.

You can also trigger skills explicitly: just describe what you need and Claude will use the right skill.

## Requirements

- Claude Code CLI or Claude Code Work (VS Code extension)
- A GitHub account with access to this repo
